Introduction to Python with Turtle Graphics

Today, we'll explore Python using the Turtle graphics module, which allows us to create graphics by controlling a "turtle" on the screen. To get started, open VS Code, create a new file named main.py, and begin by importing Turtle with import turtle. Let's set up a window for our turtle. Use the following code to create the window: 

import turtle

turtle.tracer(0)
turtle.speed(0)
turtle.hideturtle()
turtle.bgcolor('black')
turtle.title('Yay')
turtle.mainloop()

In this setup, tracer(0) and speed(0) make drawing faster, hideturtle() hides the turtle cursor, bgcolor('black') sets the background color to black, title('Yay') sets the window title, and mainloop() keeps the window open.

Next, let's draw random lines using the random module. Add the module with import random. Modify your code to include a loop that moves the turtle randomly: 

import turtle
import random

turtle.tracer(0)
turtle.speed(0)
turtle.hideturtle()
turtle.bgcolor('white')
turtle.title('Yay')
turtle.pencolor('black')
turtle.pensize(2)
turtle.goto(0, 0)

for _ in range(100):
    turtle.forward(random.randint(1, 100))
    turtle.right(random.randint(0, 360))

turtle.done()

Here, random.randint(a, b) generates a random integer between a and b, and the loop moves the turtle randomly to create abstract art. Now, let's draw a perfect circle and fill it with color. Use the following code: 

import turtle

turtle.tracer(0)
turtle.speed(0)
turtle.hideturtle()
turtle.bgcolor('white')
turtle.title('Yay')
turtle.pencolor('red')
turtle.fillcolor('red')
turtle.pensize(2)

turtle.penup()
turtle.goto(0, -100)
turtle.pendown()

turtle.begin_fill()
turtle.circle(100)
turtle.end_fill()

turtle.done()

In this code, penup() lifts the pen to move the turtle without drawing, goto(x, y) moves the turtle to specific points, circle(x, y) automates drawing points in a circle, and begin_fill() and end_fill() fill the shape drawn between them. This creates a simple flag resembling the Japanese flag.

Drawing Rectangles with Turtle

Let's learn how to draw rectangles using Turtle. We'll use the turtle.forward() and turtle.right() methods to create a rectangle. Use the following code to draw a simple rectangle: 

import turtle

turtle.tracer(0)
turtle.speed(0)
turtle.hideturtle()
turtle.bgcolor('white')
turtle.title('Rectangle')
turtle.pencolor('blue')
turtle.pensize(2)

turtle.penup()
turtle.goto(-50, 50)
turtle.pendown()

for _ in range(2):
    turtle.forward(100)
    turtle.right(90)
    turtle.forward(50)
    turtle.right(90)

turtle.done()

This code creates a rectangle with a width of 100 and a height of 50. Now, let's use this knowledge to draw the flag of Austria.

Drawing the Flag of Austria

The Austrian flag consists of three horizontal bands: red, white, and red. We'll draw this flag by stacking three rectangles on top of each other. Use the following code: 

import turtle

turtle.tracer(0)
turtle.speed(0)
turtle.hideturtle()
turtle.bgcolor('white')
turtle.title('Flag of Austria')
turtle.pensize(2)

def draw_rectangle(color, width, height):
    turtle.begin_fill()
    turtle.fillcolor(color)
    for _ in range(2):
        turtle.forward(width)
        turtle.right(90)
        turtle.forward(height)
        turtle.right(90)
    turtle.end_fill()

turtle.penup()
turtle.goto(-100, 50)
turtle.pendown()

draw_rectangle('red', 200, 50)
turtle.penup()
turtle.goto(-100, 0)
turtle.pendown()
draw_rectangle('white', 200, 50)
turtle.penup()
turtle.goto(-100, -50)
turtle.pendown()
draw_rectangle('red', 200, 50)

turtle.done()

This code defines a function draw_rectangle to draw a filled rectangle of the specified color, width, and height, and then uses it to create the flag of Austria. Functions are basically a name to cover a list of instructions. For example, if the mall, m and home, h, are 2D points, then there could be a function "Go_To_Mall_and_Back" that takes in these two inputs and moves the turtle to the mall and back. It may look like this: 

import turtle

turtle.tracer(0)
turtle.speed(0)
turtle.hideturtle()
turtle.bgcolor('white')
turtle.title('I Love The Mall')
turtle.pensize(2)

def Go_To_Mall_and_Back(h,m):

    turtle.goto(h)
    turtle.goto(m)
    turtle.goto(h)


Go_To_Mall_and_Back((0,0),(100,100))

turtle.done()
